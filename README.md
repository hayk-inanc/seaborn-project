# Seaborn Project

Проект посвящен визуализации и анализу данных с использованием библиотеки `Seaborn` и других инструментов экосистемы Python для Data Science. В рамках проекта исследуются различные типы статистических графиков, методы построения зависимостей и кастомизация визуализаций.

## Стек технологий и библиотеки

В проекте используются следующие ключевые инструменты:

* **Python 3.8+** — основной язык разработки.
* **Seaborn** — библиотека для создания эстетичных и информативных статистических графиков.
* **Matplotlib** — база для визуализации, используемая для тонкой настройки элементов графиков.
* **Pandas** — загрузка, обработка и структурирование данных в виде DataFrame.
* **NumPy** — математические вычисления и работа с многомерными массивами.
* **Jupyter Notebook** (опционально) — интерактивная среда для запуска и анализа кода.

## Архитектура проекта

Основные направления визуализации, представленные в репозитории:
* **Relational plots** (Связи между непрерывными переменными).
* **Categorical plots** (Анализ категориальных признаков).
* **Distribution plots** (Распределения данных, гистограммы).
* **Multi-plot grids** (Использование `FacetGrid` для построения сложных матриц графиков).
